Calculadora em JS

Calculadora criada com HTML, CSS e Javascript. Para utilizar, clique nos botões na tela ou use o teclado de seu computador. Além dos número e operadores, os outros botões do teclado são:

  • ESC = Zerar calculadora.
  • ENTER = Mostrar resultado.
  • BACKSPACE ou DEL = Apagar último valor inserido.
